# Basic Laravel JSON API

## CRUD requests:
1. Articles :
- GET /api/articles
- GET /api/articles/{id}
- POST /api/articles
- PUT /api/articles/{id}
- DELETE /api/articles/{id}
2. Users:
- GET /api/users
- GET /api/users/{id}
- POST /api/users
- PUT /api/users/{id}
- DELETE /api/users/{id}

## JSON fields:
1. Article:
```
{
    "title": "The title",
    "body": "The body"
}
```
2. User:
```
{
    "name": "The name",
    "email": "The e-mail",
    "password": "The password"
}
```
